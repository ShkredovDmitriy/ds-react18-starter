# ds-react18-starter

Create-react-app based template for quick website development

#### Libraries

- React 18
- Typescript
- SCSS

#### Structure

```
build/                # build files
public/               # static files
src
  |_ blocks/          # header, footer, section
  |_ components/      # nav, ul
  |_ elements/        # a, span, button
```

## Installation
Please use Nodejs 16 or newer

```
npm install
```

#### Start

```
npm run start
```

#### Build

```
npm run build
```
